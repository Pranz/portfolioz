Support Group Network (School Project)

WE WERE COMMISSIONED BY A COMPANY CALLED SUPPORT GROUP NETWORK (SGN) TO UPDATE THEIR EXISTING WEBSITE. THERE WAS A LOT TO CHANGE, SO ME AND MY CLASSMATE (FANNY VICKIUS) DECIDED THAT WHAT WOULD GIVE THE BEST RESULTS WOULD BE TO RECREATE THE WEBSITE FROM SCRATCH, WE UPDATED THE EXISTING IDENTITY, GAVE SUGGESTIONS ON FUNCTIONS ON THE WEBSITE AND REDESIGNED THE WEBSITE. SUPPORT GROUP NETWORK IS AN ORGANIZATION THAT WORKS TO STRENGTHEN THE LINKS BETWEEN ASYLUM SEEKERS AND SWEDISH CITIZENS. A WAY FOR ASYLUM SEEKERS TO GET INVOLVED WITH THE SWEDISH SOCIETY WHILE WAITING FOR A RESIDENCE PERMIT AND A WAY FOR OUTSIDERS TO BE ABLE TO CONTRIBUTE IN ANY FORM.
